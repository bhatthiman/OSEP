# OSEP

Open Static Equipment Platform

An open-source engineering platform for static equipment design, engineering calculations, CAD generation, STEP export, and future analysis integration.

## Vision

Build a modular engineering ecosystem for:
- Pressure vessels
- Heat exchangers
- Columns
- Storage tanks
- Engineering calculations
- STEP generation
- Drawing generation
- CFD/FEA integration
